# Projecte: Gestio de Parelles Ling?istiques

Aplicacio web academica per gestionar parelles linguistiques entre voluntaris i aprenents de catala.

## Objectiu
Crear una base simple, clara i mantenible per treball en equip amb metodologia Agile.

## Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Node.js + Express
- Base de dades: MySQL
- Control de versions: Git

## Estructura
- `frontend/`: HTML/CSS/JS estatics
- `backend/`: API amb Express (MVP)
- `database/`: schema SQL i seeds
- `docs/`: documentacio del projecte

## EPICS
1. Homepage
2. Gestio d'usuaris, autenticacio i acces
3. Panell de gestio de parelles linguistiques
4. Documentacio del projecte

## Comencar
### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`
4. Healthcheck: `GET http://localhost:3000/api/health`

### Frontend
Obrir `frontend/index.html` amb un navegador.
